# Playwright Skill

## Description
Automated testing of Exocortex surfaces using Playwright.

## Capabilities
- Cross-surface testing (web, mobile, Telegram)
- End-to-end validation
- Fault injection testing
- Performance monitoring
- Visual regression testing

## Usage
- `test heronclient web interface`
- `test cross-surface consistency`
- `inject fault and verify recovery`
- `generate test report`

## Configuration
- Browser: Chromium, Firefox, WebKit
- Devices: Desktop, mobile
- Authentication: Test accounts
- Traces: Enabled for debugging
